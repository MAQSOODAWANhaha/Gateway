# Web 设计规范（Gateway 控制台）

## 风格定位
- 调性：偏 Nginx Proxy Manager 的高密度运维后台风格
- 视觉目标：卡片状态条 + 强对比分割线 + 高可读性
- 关键记忆点：蓝紫主色 + 青色/橙色辅助 + 多层次灰阶背景

## 主题体系
- 默认主题：Light
- 可选主题：Dark
- 高对比：可选（辅助可访问性）

## 字体
- 标题：`Noto Sans SC`
- 正文：`Noto Sans SC`
- 原则：统一、清晰、兼顾中英混排

## 背景与层级
- 背景采用低饱和径向渐变，保留运维感与专业感
- 卡片顶部状态条用于强调模块类型
- 分割线比普通边框更强，形成清晰表格层级

## 导航与布局
- 侧栏支持折叠与展开（桌面）
- 手机端侧栏抽屉式弹出
- 活跃项使用背景高亮 + 左侧色条 + 图标底色区分
- 侧栏结构参考 shadcn/ui Sidebar 组件（分组、菜单、底部区）

## 组件规范
- 卡片：圆角 14px，状态条 + 明显分割线 + 轻阴影
- 按钮：主色实心 / 描边 / 柔和底三种
- 表格：表头底色 + 斑马纹 + 强分割线

## 交互
- Hover/Focus 清晰可见
- 输入控件最小高度 36px（更高密度）
- 可点击元素统一指针样式
